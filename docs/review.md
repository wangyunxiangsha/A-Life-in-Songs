## 框架现状（只读审查）

### 优点

| 项 | 说明 |
|----|------|
| 配置集中 | 内容集中在 `config.ts`，适合人生故事批量注入 |
| 章节弹性 | `chapters.length` 驱动卡片布局，支持 18 章扩展 |
| 类型导出 | `Chapter` 类型由 config 推导，改章节结构有类型提示 |
| 音频容错 | 空 `audio` 时 UI 禁用播放，不报错 |

### 定制前需关注的问题

| 严重度 | 问题 | 位置 | 建议 |
|--------|------|------|------|
| 中 | 区块标题与默认绘本绑定 | `Seasons.tsx`、`Companions.tsx` | 改为人生主题或迁入 config |
| 中 | 时间滑块标签为梦境语义 | `TimeSlider.tsx` | 改为晨/昼/暮/夜等 |
| 低 | 18 章无导航 | 全站 | 后续可加章节索引 |
| 低 | CSS 类名 `glass-panel-${id}` | 全局样式 | 新 id 需确认是否有对应样式，或依赖内联 style |
| 信息 | 根 README 端口写 4000 | `README.md` | vite.config 为 3000，以实际为准 |

### 18 章规模评估

- **性能**：18 个视频若同时预加载可能占内存；需确认 `VideoBackground` 是否单实例切换（当前为切换 src，可接受）
- **体验**：滚动距离长，符合「一生」叙事，但首次访客可能迷失；后续可考虑进度条
- **布局**：Companions 18 张卡片横排，小屏可能拥挤——属框架既有行为，定制后需实机查看
