# 十八 Chapter 角色卡片

> 叙事主角为 **东方波**；角色卡按章呈现该段生命里最重要的人（或东方波本人）。  
> 图片路径先用占位符，语音暂空。可直接填入 `config.ts` 的 `character` 字段。

## 人物出场一览

| 人物 | 出现章节 | 关系 |
|------|----------|------|
| **小梦** | 1、2、4 | 同班青梅，六一领舞；小学陪练；初中心动的大小姐 |
| **阿驰** | 5、6 | 好友，大话西游3；KTV 唱《月亮惹的祸》 |
| **瑶瑶** | 9 | 复读结束夏天，聚会唱歌 |
| **小晴** | 13、14 | 大学看电影；大四图书馆考研同伴 |
| **小洁** | 15 | 女友，复读相识至研究生 |
| **娣总** | 18 | 移植仓内陪伴的护士 |
| **东方波** | 3、7、8、10、11、12、16、17 | 其余章节以「我」为卡片主角 |
