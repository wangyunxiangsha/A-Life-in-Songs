> 画风与《AI人物设定-插画绘本.md》《AI分镜提示词-插画绘本.md》一致。  
> **用法**：ChatGPT 生图时先上传「参考图」，再粘贴对应章节整段提示词。  
> **输出**：3:4 竖图 → 存入 `public/images/character/` → 复制为 `public/images/chXX.png` → 更新 `config.ts`。

## 一、通用操作

### 1. 参考图怎么选

| 情况 | 上传什么 |
|------|----------|
| 该角色**第一次**出图 | `public/images/ref/` 里对应该人的照片（见下表） |
| 该角色**已有满意图** | 上一张同角色角色卡（保证跨章同脸） |
| 小梦 ch01–04 | ✅ 已完成，ch02/ch04 垫 ch01 |

### 2. 每条提示词固定结构

```
【角色锁定】+ 场景服装 + 构图 3:4 + 画风锚点 + 章节色调 + 负面约束
```

### 3. 画风锚点（每条末尾已包含，勿删）

```
插画绘本风格，柔和水彩晕染，淡彩铅笔线稿，温暖叙事感，
中国九十年代至当代青春回忆录美学，非动漫，非写实照片，
画面干净留白，情绪细腻克制，轻微纸张纹理。
3:4 竖构图角色卡，人物居中，背景柔和虚化 bokeh。
```

### 4. 负面提示（生成不对时追加一句）

```
不要换脸、不要动漫大眼、不要写实照片、不要畸形手指、
不要文字水印 logo、不要血腥惨烈医疗画面。
```

### 5. 推荐生成顺序（减少「不像同一人」）

```
✅ 小梦 ch01 → ch02 → ch04（已完成）
① 东方波 ch03（初中基准）→ ch07（高三基准）→ ch08、10、11、12、16、17
② 阿驰 ch05 → ch06
③ 瑶瑶 ch09
④ 小晴 ch13 → ch14
⑤ 小洁 ch15
⑥ 娣总 ch18
```

### 6. 文件命名

```
public/images/character/{角色}-{章节关键词}-1.png
→ 复制为 public/images/chXX.png
```
