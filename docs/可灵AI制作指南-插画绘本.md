> 平台：[可灵 AI](https://klingai.kuaishou.com/)（快手）  
> 配套提示词：`AI人物设定-插画绘本.md`、`AI分镜提示词-插画绘本.md`  
> 目标产出：19 段视频 + 18 张角色卡 → `public/videos/`、`public/images/`

---

## 一、整体流程（推荐）

```
可灵「AI图片」定画风 → 出 7 张角色基准图
        ↓
可灵「AI图片」+ 角色参考 → 出 18 张角色卡 ch01–ch18
        ↓
可灵「AI图片」→ 出 19 张视频关键帧（16:9，含首屏）
        ↓
可灵「图生视频」→ 每章 5s 或 10s 微动效
        ↓
剪映拼接/循环 → 导出 chXX.mp4（约 15s，<10MB）
```

**原则**：静态图用「文生图」控画风；视频用「图生视频」控运动。**图生视频不要重复描述外貌**，只写怎么动。

## 三、文生图设置（角色卡 + 关键帧）

### 通用参数

| 项 | 建议 |
|----|------|
| 风格 | 插画 / 绘本 / 水彩（选最接近的一项，或靠提示词约束） |
| 品质 | 先「标准」试词，满意后「高品质」出最终版 |
| 角色卡比例 | **3:4** |
| 视频关键帧比例 | **16:9** |
| 清晰度 | 尽量高清，主体居中，留白充足 |

### 提示词拼装公式

```
【当章画面描述】（来自 AI分镜提示词-插画绘本.md）
+ 【角色名与特征】（来自 AI人物设定-插画绘本.md，角色章节必加）
+ 【全局风格锚点-中文】
+ 【角色卡后缀 或 关键帧后缀】
```

**示例（第 1 章角色卡 · 小梦）**
```
小梦在小学礼堂舞台上领舞，黑色马尾，红领巾白衬衫，
双臂舒展，聚光灯暖杏色，台下模糊观众席，笑容自信明亮，
插画绘本风格，柔和水彩晕染，淡彩铅笔线稿，温暖叙事感，
中国青春回忆录美学，非动漫非写实，干净留白，
竖版人物插画，3比4构图，柔和背景虚化，无文字无水印
```

### 角色一致性（可灵）

1. 先用基准提示词生成 **小梦 / 东方波 / 阿驰…** 各 1 张满意图  
2. 后续文生图时 **上传该图作为参考图**（或「角色特征」）  
3. 提示词写：**保持与参考图同一人物，同一画风，只改变服装与场景**  
4. 小梦：ch01 满意 → 参考图做 ch02、ch04  
5. 小晴：ch13 满意 → 参考图做 ch14  
6. 东方波：ch07 满意 → 参考图做 ch03、08、10–12、16、17  

### 文生图负面提示（可灵「不希望出现的内容」）

```
写实照片，3D渲染，动漫，畸形手指，多余肢体，脸部变形，
文字，水印，LOGO，低清晰度，血腥，恐怖，过度惨白的病房
```

## 四、图生视频设置（背景视频）

### 参数建议

| 项 | 建议 |
|----|------|
| 模式 | **图生视频**（上传对应章关键帧） |
| 画幅 | **16:9** |
| 时长 | 网站循环用：**5 秒** × 3 段拼接 ≈ 15 秒；或单次 **10 秒** 再剪 |
| 品质 | 关键章用「高品质」，批量测试用「标准」省积分 |
| 背景音乐 | **关闭**（网站自有 BGM） |

### 可灵图生视频提示词公式

> **不要写人物长什么样**，只写运动与镜头。

```
镜头语言 + 主体运动 + 背景运动 + 氛围 + 稳定性约束
```

**通用模板（插画绘本场景）**
```
保持插画绘本水彩画风与构图不变，画面稳定不变形。
镜头缓慢向前推进；【主体微动，如灯光摇曳/旗帜轻摆/树叶飘落】；
背景【光影轻微变化】；氛围温暖怀旧；无镜头切换，无人物变脸。
```

### 图生视频负面描述

```
画面变形，风格突变，镜头跳切，快速变焦，人物变脸，
多余手指，闪烁，抖动，写实化，3D化，文字模糊，水印
```

### 人物出现在关键帧时（如 ch01 舞台）

额外加上：
```
保持人物五官、发型、服装与参考图一致，动作幅度小，
自然眨眼或微笑，禁止夸张扭动，禁止脸部变形
```

## 五、导出与放入项目

### 角色卡

1. 可灵下载 **3:4 图** → 重命名为 `ch01.png` … `ch18.png`  
2. 放入 `e:\git-code\Dreamways\public\images\`  
3. 建议边长 ≥ 800px，JPG 质量 80%+  

### 背景视频

1. 可灵导出 **16:9 MP4**  
2. 剪映：3 段 5s 拼接为 15s，首尾加 0.5s 淡化便于循环  
3. HandBrake 压缩：H.264，1280×720，目标 **< 10MB**  
4. 命名 `首屏.mp4`、`ch01.mp4` … `ch18.mp4`  
5. 放入 `e:\git-code\Dreamways\public\videos\`  

### 本地验证

```bash
npm run dev
```

滚动各章 → 背景应切换 → 点播放 BGM 时背景视频暂停（已实现）。

## 九、常见问题

**Q：图生视频后画风变写实了？**  
A：关键帧不够「插画画风」或运动词太复杂。重新出关键帧，加强水彩绘本词；图生视频写「保持插画绘本水彩画风不变」。

**Q：人物脸崩？**  
A：换更清晰正面关键帧；运动改为「固定镜头 + 背景动」；人物只写「轻微眨眼/呼吸」。

**Q：5 秒不够长？**  
A：可灵生成 5s，剪映复制三段做 15s 循环，比一次生成 10s 更省积分且更易循环。

**Q：可灵文生图和 Midjourney 混用？**  
A：可以。角色卡用 Midjourney 出，视频关键帧统一画风后导入可灵图生视频即可。
