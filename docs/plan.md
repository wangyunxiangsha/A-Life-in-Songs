# 需求与项目计划 · 我的一生

## 项目概述

基于 Dreamways 沉浸式网站框架，将默认绘本故事「梦途 / 卢卡」定制为个人人生回忆录网站：**十八首歌，十八段人生经历**。访客通过滚动浏览章节，配合背景视频、背景音乐与角色卡片，沉浸式回顾从童年到当下的成长轨迹。
