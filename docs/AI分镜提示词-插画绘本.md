## 图生视频参数建议

| 参数 | 建议 |
|------|------|
| 时长 | 生成 5s × 3 段拼接，或一次 10s，剪到 **15 秒** |
| 运动 | `slow push in` / `gentle breeze` / `subtle light flicker` |
| 循环 | 首尾帧构图、光线接近；剪映加 0.5s 交叉淡化 |
| 导出 | H.264，1280×720，< 10MB |
