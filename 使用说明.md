# 梦途 Dreamways · 使用说明

这是一个可以自由定制内容的网站框架。你只需要替换资源文件、修改一个配置文件，就能做出属于自己的版本。

---

## 第一步：替换资源文件

把你自己的文件放到 `public/` 下对应的文件夹里：

| 文件夹 | 放什么 | 格式 |
|--------|--------|------|
| `public/videos/` | 背景视频 | `.mp4` |
| `public/audio/` | 背景音乐、角色语音 | `.mp3` |
| `public/images/` | 主角照片、角色图片 | `.jpg` / `.png` |

文件名可以自己起，后面在配置文件里填对应的文件名就行。

---

## 第二步：修改配置文件

打开 `src/content/config.ts`，这是**唯一需要修改的文件**。

---

### 一、网站基本信息

```ts
export const siteConfig = {
  title: '梦途',                // 网站大标题（首屏、导航栏）
  titleSub: 'Dreamways',       // 英文副标题（导航栏 logo 旁）
  heroSubtitle: '四季四梦，每夜启程。', // 首屏一句话副文案
  scrollHint: '向下滚动',       // 右侧滚动提示文字
  footerQuote: '「每次闭上眼睛，就是一次新的出发」', // 页脚引用语
  copyright: '2026 @哈维yao',  // 页脚版权信息
  introVideo: '/videos/intro.mp4', // 首屏背景视频
};
```

---

### 二、主角介绍区块

```ts
export const protagonist = {
  show: true, // 不需要这个区块？改成 false，整块从页面消失

  sectionTitle: '关于卢卡',            // 区块标题
  image: '/images/luka-portrait.jpg', // 主角照片（public/images/ 下）
  imageLabel: '卢卡，五岁半',           // 照片下方标注文字
  intro: '卢卡五岁半……',              // 介绍正文
  audio: '/audio/luka-intro.mp3',    // 语音音频（没有就填空字符串 ''）
};
```

---

### 三、章节列表

**每个章节同时控制两处内容：**
- 「四季之梦」里的故事滚动屏
- 「卢卡的伙伴们」里的角色卡片

**增加章节**：把文件末尾的注释模板取消注释，填入内容即可。  
**减少章节**：直接删掉对应的 `{ ... }` 整块。  
卡片布局会自动适配章节数量，无需手动调整。

每个章节包含以下字段：

```ts
{
  // ── 基本信息 ──────────────────────────────────────────
  id: 'spring',         // 唯一 ID，只用英文字母和连字符，不能重复
  video: '/videos/spring-sky.mp4',  // 章节背景视频（public/videos/ 下）
  title: '第一章 · 春 · 入口',       // 章节标题
  story: '游乐场的入口……',          // 故事正文
  audio: '/audio/spring-dream.mp3', // 章节背景音乐（public/audio/ 下）
  color: '#FAD0C4',                 // 主题色，影响边框和高亮颜色
  glassBg: 'rgba(250, 208, 196, 0.12)', // 故事面板背景色，保持低透明度

  // ── 角色卡片 ──────────────────────────────────────────
  character: {
    name: '豆豆',                          // 角色名字
    role: '种子精灵',                       // 角色称号
    emoji: '🌱',                           // 图片加载失败时的备用表情
    image: '/images/char-spring.jpg',      // 角色图片（public/images/ 下）
    visualGradient: 'linear-gradient(160deg, #fce4ec 0%, #f8bbd0 45%, #ede7f6 100%)',
                                           // 卡片图片区的渐变底色
    desc: '住在游乐场入口那棵老树的树根后面。', // 卡片正面一句话简介
    detail: '豆豆是……',                    // 卡片背面详细介绍（翻转后显示）
    intro: '我是豆豆……',                   // 角色自我介绍（卡片背面展示）
    audio: '/audio/bud-intro.mp3',        // 角色语音（public/audio/ 下，没有就填 ''）
  },
}
```

## 常见操作

### 隐藏主角介绍区块

```ts
protagonist: {
  show: false, // 改这一行就够了
  ...
}
```

### 新增一个章节

把 `config.ts` 文件末尾的注释模板取消注释，修改成你自己的内容：

```ts
{
  id: 'rainy-day',           // 唯一 ID
  video: '/videos/rain.mp4',
  title: '第五章 · 雨天',
  story: '那天下雨了……',
  audio: '/audio/rain-dream.mp3',
  color: '#6fa8dc',
  glassBg: 'rgba(111, 168, 220, 0.12)',
  character: {
    name: '雨滴',
    role: '小水滴',
    emoji: '💧',
    image: '/images/char-rain.jpg',
    visualGradient: 'linear-gradient(160deg, #e3f2fd 0%, #bbdefb 100%)',
    desc: '住在屋檐下的第三滴水。',
    detail: '详细介绍……',
    intro: '我是雨滴……',
    audio: '/audio/raindrop-intro.mp3',
  },
},
```

### 删除一个章节

在 `chapters` 数组里找到对应的 `{ ... }` 对象，整块删掉即可。

### 没有音频怎么办

把音频字段填成空字符串，播放按钮会自动变为不可用状态，不会报错：

```ts
audio: '',
```

---

## 启动项目

双击根目录里的 **`启动.command`**（Mac）或 **`启动.bat`**（Windows）。

或者在终端里运行：

```bash
npm run dev
```

然后在浏览器里打开 `http://localhost:5173` 即可预览。

## 文件结构总览

```
Dreamways/
├── public/
│   ├── videos/       ← 背景视频放这里
│   ├── audio/        ← 音频文件放这里
│   └── images/       ← 图片文件放这里
│
├── src/
│   └── content/
│       └── config.ts ← ★ 唯一需要修改的文件
│
├── 使用说明.md        ← 本文件
├── 启动.command      ← Mac 启动脚本
└── 启动.bat          ← Windows 启动脚本
```

`src/` 下的其他所有文件（组件、样式、逻辑）**不需要修改**。
